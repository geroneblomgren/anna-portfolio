  <name>Task 2: Install Tailwind v4 with dark design tokens and contrast-verified color palette</name>
  <files>src/app/globals.css, postcss.config.mjs</files>
  <action>
1. Check if Tailwind is already installed and what version. If v3 is present (indicated by a tailwind.config.ts/js file), run `npx @tailwindcss/upgrade` to migrate to v4. If no Tailwind, install fresh: `npm install tailwindcss @tailwindcss/postcss postcss`.

2. Delete `tailwind.config.ts` or `tailwind.config.js` if present (v4 uses CSS-first config).

   ```javascript

     },
   }
   ```

4. Create or replace `src/app/globals.css` with the full design token system:
   ```css
   @import "tailwindcss";

   @theme {
     /* Color system — warm dark palette */
     --color-bg: #1a1614;
     --color-surface: #252220;
     --color-border: #3d3633;

     /* Text */
     --color-text-heading: #f0ebe6;
     --color-text-body: #d4ccc4;
     --color-text-muted: #8a7e76;
     --color-text-muted-on-surface: #9a8e86;

     /* Accent — amber/gold */
     --color-accent: #c8956c;
     --color-accent-hover: #d4a67d;
     --color-accent-muted: #8b6b4a;

     /* Functional */
     --color-error: #d45e4d;
     --color-success: #7a9a6e;

     /* Typography */
     --font-heading: var(--font-bodoni-moda), Georgia, serif;
     --font-body: var(--font-dm-sans), system-ui, sans-serif;
   }
   ```

   IMPORTANT contrast adjustments (per research):
   - `--color-text-muted-on-surface: #9a8e86` — new token for muted text on surface backgrounds (~4.8:1 ratio, passes AA). Use `text-text-muted-on-surface` class when text is on `bg-surface`.
   - `--color-error: #d45e4d` — adjusted from #c05545 to achieve ~4.7:1 on background (passes AA for body text). Per user decision, #c05545 was the starting point but contrast verification requires adjustment.
   - Original muted text #8a7e76 is kept for use on the primary background (4.56:1, passes AA).

5. Verify Tailwind generates utility classes by checking that the dev server compiles without errors and `bg-bg`, `text-accent`, `font-heading` classes are recognized.
