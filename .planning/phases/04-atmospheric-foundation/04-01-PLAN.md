Add two CSS blocks to globals.css (after the existing `.tag-filter-scroll` rules, before the `@theme` block):

**1. Vignette overlay on `body::before`:**
```css
body::before {
  content: '';
  position: fixed;
  inset: 0;
  pointer-events: none;
  z-index: 9998;
  background: radial-gradient(
    ellipse at center,

    rgba(0, 0, 0, 0.45) 100%
  );
}
```

**2. Film grain overlay on `body::after`:**
```css
body::after {
  content: '';
  position: fixed;
  inset: 0;
  pointer-events: none;
  z-index: 9999;
  opacity: 0.035;
  background-image: url("data:image/svg+xml,<svg xmlns='http://www.w3.org/2000/svg'><filter id='g'><feTurbulence type='fractalNoise' baseFrequency='0.65' numOctaves='3' stitchTiles='stitch'/></filter><rect width='100%' height='100%' filter='url(%23g)'/></svg>");
  animation: grain 0.2s steps(1) infinite;
}

@keyframes grain {
  0%, 100% { transform: translate(0, 0); }
  20%  { transform: translate(-2%, -3%); }
  40%  { transform: translate(2%,  2%); }
  60%  { transform: translate(-1%, 3%); }
  80%  { transform: translate(3%, -1%); }
}
```

**3. Reduced-motion guard:**
```css
@media (prefers-reduced-motion: reduce) {
  body::after {
    animation: none;
    opacity: 0.02;
  }
}
```

**4. Add `isolation: isolate` to the masonry grid container** to contain GPU compositing:
```css
.masonry-grid {
  /* existing rules... */
  isolation: isolate;
}
```

Important implementation notes:
- `pointer-events: none` is mandatory on both overlays — without it, the entire site becomes unclickable
- Grain z-index 9999 (topmost visual), vignette z-index 9998 (below grain, above content)
- Grain opacity 0.035 — do NOT exceed 0.05 (reads as dirty noise on OLED)
- The `isolation: isolate` on masonry-grid prevents implicit compositing explosion from the high-z-index grain overlay
