**1. Intro Animation (first visit)**
- Open http://localhost:3000 in an incognito/private window

- Close incognito window, open new one — intro should play again (fresh localStorage)

**2. Reduced Motion**
- In browser devtools, enable "Prefer reduced motion" (Rendering tab)
- Open http://localhost:3000 in incognito — should go straight to gallery, no animation

**3. Gallery & Tag Filtering**
- Verify: art pieces display in a masonry grid (multiple columns on desktop, single column on narrow mobile)

- Click "Drawings" pill — only drawings should remain visible
- Click "All" — all pieces return
- Verify: featured pieces (if any) appear first in the grid

- Click any gallery image — full-screen lightbox should open
- Verify: high-resolution image displayed (lightbox 2400px variant)
- Verify: title, medium, description, and tags are shown
- Use arrow keys (left/right) to navigate between pieces
- Try swipe gestures on mobile/touch

**5. Navigation**

- Click hamburger — overlay menu should show all three links
- Click a link — should navigate and close menu
