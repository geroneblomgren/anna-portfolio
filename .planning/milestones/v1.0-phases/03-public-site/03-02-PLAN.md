  <action>
1. Create `src/components/frontend/IntroAnimation.tsx` as a `'use client'` component:
   - Props: `{ children: React.ReactNode }`
   - State: `showIntro` (boolean, default `false` — avoids SSR hydration mismatch), `done` (boolean, default `false`)
   - `useEffect` on mount:
     - Check `localStorage.getItem('intro-seen')`
     - Check `window.matchMedia('(prefers-reduced-motion: reduce)').matches`
     - If not seen AND no reduced motion: `setShowIntro(true)`
     - Otherwise: `setDone(true)` (skip straight to gallery)
   - `skip()` function: sets localStorage `'intro-seen'` to `'1'`, sets `showIntro(false)`, sets `done(true)`
   - Auto-complete: after animation finishes (~3.5s), call `skip()` automatically via `onAnimationComplete` or `setTimeout`

   Animation design (ink-bleed style matching Anna's B&W charcoal/ink art):
   - Use `motion` from `motion/react` for all animations
   - Use `AnimatePresence mode="wait"` to handle intro→gallery crossfade
   - Full viewport overlay (`fixed inset-0 bg-bg z-50 flex items-center justify-center`)

   SVG ink strokes:
   - 3-4 organic SVG `<motion.path>` elements simulating ink strokes spreading across the screen
   - Each path uses `pathLength` animation: `initial={{ pathLength: 0, opacity: 0 }}` → `animate={{ pathLength: 1, opacity: 1 }}`
   - Stagger paths with increasing `delay` (0s, 0.3s, 0.6s, 0.9s)
   - Stroke color: `#c8c8cc` (text-body) with `strokeWidth: 2-3`, `fill: none`
   - SVG paths should be wide, sweeping Bezier curves that suggest ink flowing across paper — use `Q` and `C` path commands for organic feel
   - SVG viewBox: `0 0 800 400` centered in the viewport

   Text reveal (after strokes):
   - "Anna Blomgren" in `font-heading text-text-heading text-5xl md:text-7xl` — fades in at ~1.5s delay
   - "Artist & Illustrator" in `font-heading text-accent text-xl italic` — fades in at ~2s delay
   - Both use `motion.div` with `initial={{ opacity: 0, y: 20 }}` → `animate={{ opacity: 1, y: 0 }}`

   Exit transition:
   - The entire intro `motion.div` has `exit={{ opacity: 0 }}` with `transition={{ duration: 0.8 }}`
   - Gallery `motion.div` has `initial={{ opacity: 0 }}` → `animate={{ opacity: 1 }}` with `transition={{ duration: 0.6, delay: 0.2 }}`

   Skip button:
   - Small "Skip" text in bottom-right corner: `fixed bottom-6 right-6 text-text-muted text-sm font-body hover:text-accent transition-colors cursor-pointer z-50`
   - Only shows during intro (inside the intro motion.div)
