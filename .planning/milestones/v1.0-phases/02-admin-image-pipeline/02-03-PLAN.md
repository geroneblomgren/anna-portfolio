  <how-to-verify>
Start the dev server (`npm run dev`) and open http://localhost:3000/admin

**1. Authentication (ADM-01):**
- If first visit, create an admin user at the registration screen
- Log out and try accessing /admin — should redirect to login
- Log in with valid credentials — should grant access
- Try wrong password — should be rejected

**2. Art Piece Upload (ADM-02, INF-01):**
- Click "Art Pieces" in sidebar → "Create New"
- Fill in title, select tags (e.g., "Drawings"), add medium and description
- Upload a JPEG image (any size, ideally a camera photo or scan)
- Save the piece
- Verify the admin shows a thumbnail of the processed image
- Check that the image preview loads (this confirms WebP processing worked)

**3. Edit and Delete (ADM-03, ADM-04):**
- Edit the piece you just created — change the title, replace the image
- Save and verify changes persist after page reload
- Create a second piece, then delete it
- Verify only the first piece remains

**4. Reorder and Featured (ADM-07):**
- Create 2-3 art pieces
- In the art pieces list view, try dragging to reorder them
- Toggle "Featured" on one piece — verify it appears first in the list
- If you mark 6+ pieces as featured, a warning should appear

**5. About Section (ADM-05):**
- Click "About" in sidebar globals
- Edit bio text, upload a photo, write an artist statement
- Add contact email and phone
- Save and verify all fields persist

**6. Contact & Social Links (ADM-06):**
- Click "Site Settings" in sidebar
- Add a social link (e.g., platform: "Instagram", url: "https://instagram.com/anna")
- Save and verify it persists

**7. QR Code (ADM-08):**
- Click "QR Code" in the admin sidebar navigation
- Verify a QR code image appears with "Dark Arts by Anna" branding
- Verify the QR uses the dark/cold graphite aesthetic (light code on dark background)
- Click the download button — verify a PNG file downloads
- Scan the QR code with your phone — verify it opens the correct URL
- Update the QR URL in Site Settings, return to QR page — verify URL updates

**8. Image Quality Check (INF-01):**
- For any uploaded art piece, check that the image loads quickly
- In browser DevTools Network tab, verify the served image is WebP format
- File size should be reasonable (under ~300KB for gallery variant)
