# Anna Blomgren — Artist Portfolio

## What This Is

An interactive portfolio web app for Anna Blomgren, an accomplished mixed-media artist seeking a tattoo apprenticeship. The site replaces the traditional binder/email portfolio with a stunning, always-accessible digital showcase that Anna can manage herself. Designed to impress tattoo shop owners and artists when they scan a QR code Anna hands them.

## Core Value

When someone scans Anna's QR code, the site must make an immediate, powerful visual impression that communicates: she's versatile, she has a personal voice, and she's dead serious about this career.

### Out of Scope

- E-commerce / selling art — this is a portfolio, not a shop
- User accounts / registration — only Anna has admin access
- Blog / news section — focus is purely on the work
- Booking system — contact form is sufficient for initial outreach

## Context

- Anna is an eclectic artist working across multiple media: pencil, paint, digital, tattoo designs
- The primary audience is tattoo shop owners evaluating her for apprenticeship
- Secondary audience is anyone she hands a QR code to (events, networking, social)
- The site will often be first-viewed on mobile via QR code scan
- No existing domain or hosting — needs deployment recommendation
- The design should feel like a curated gallery experience, not a generic portfolio template
- Dark/moody aesthetic matches the tattoo industry vibe

## Constraints

- **Hosting**: Needs a free or low-cost hosting solution (starting from scratch)
- **Admin simplicity**: Anna is the only user — admin should be dead simple, not a CMS
- **Performance**: Images are the product — they must load fast and look sharp on all devices
- **Mobile-first**: QR code → phone is the primary entry path
